Searchfriend is a plugin to help manage multi-shipping address.
It included:
1) Back-end: Manage orders, users, custom address(no user), order logs
2) Front-end: following are shortcodes to manage order, address
 2.1) [searchfriend_manage_friends]: This shortcode to make view manage friends (add, make request, etc)
 2.2) [searchfriend_manage_request_friends]: This shortcode to make view manage friend requests(Manage, accept/Reject)
 2.3) [searchfriend_manage_orders]: This shortcode to make view manage orders
 2.4) [searchfriend_manage_custom_friends]: This shortcode to make view manage custom address(Not as an user)
 2.5) [searchfriend_manage_send_activitylogs]: This shortcode to make view manage activities shipping logs send
 2.6) [searchfriend_manage_receive_activitylogs]: This shortcode to make view manage activities shipping logs receives
 2.7) [searchfriend_shipping_product_getting]: This shortcode to make view receive product by friend sent

Note: All shortcode workings

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress 
3. Check shortcodes to create views and use it
